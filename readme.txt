In order to use this program you need to:

1. Open the config.py with a text editor
2. Change the lethal_company_install_directory to your own
3. Adjust the thunderstore_mod_urls as needed
4. Save and close the config.py
5. Double click the run-mod-updater.exe

For your safety, this program also creates backups of your current BepInEx folder before modifying it.
